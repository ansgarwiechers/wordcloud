A [Blosxom][1] plugin for generating a tag cloud from page keywords. Requires
the `find` and `keywords` plugins (included). Also requires a patched version of
the [`HTML::TagCloud`][2] Perl module for case-insensitive sorting (patch
included).

[1]: http://blosxom.sourceforge.net/
[2]: https://metacpan.org/pod/HTML::TagCloud
